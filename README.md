# superdisplay
Started: Feburary 27th, 2026  
Last Updated: April 1st, 2026  

## About
superdisplay is my second attempt at building a barebones rendering pipeline, and it is the first project I am uploading with GitHub. This project is being written in C++ with help from the SDL3 development library for its user input and window/rendering capabilities.

## Installation
If you want to look at the source code, download `superdisplay_stable1_source.zip`, otherwise, download `superdisplay_stable1.zip`.

When you run the .exe for the first time, Windows Defender might pop up and prevent you from running, as the program is from an unknown publisher. That is because I do not have a signature on it.  

All the .dll files should be included in the directory of the .exe. If there is one missing please let me know.  

For the source code, it is all available in the src and include folders in the `superdisplay_stable1_source.zip`.
